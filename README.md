# Advent of Code Solutions in R7RS Scheme
## Author 
[Jef Grupping](https://www.github.com/jefgr)
## Overview
This repository contains my solutions for the [Advent of Code](https://adventofcode.com/) programming challenges, implemented using R7RS Scheme.

## Repository Structure
```
.
├── day1.rkt
├── day2.rkt
├── {...}
|
└── README.md
```

## Scheme Implementation
These solutions are written in R7RS Scheme.
I do use private R7RS Library, which I am not able to share.
